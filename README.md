# generator-miruken

> [Yeoman](http://yeoman.io) generator

## Getting Started

### Install Yeoman?

```bash
npm install -g yo
```

### Yeoman Generators

To install generator-miruken from npm, run:

```bash
npm install -g generator-miruken
```

Finally, initiate the generator:

```bash
yo miruken
```
